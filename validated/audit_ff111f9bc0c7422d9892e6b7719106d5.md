### Title
`MaturityVerifier` Skips Cellbase Immaturity Check for `resolved_dep_groups` — (`verification/src/transaction_verifier.rs`)

### Summary

`MaturityVerifier::verify()` enforces the cellbase maturity rule on `resolved_inputs` and `resolved_cell_deps`, but silently omits the same check for `resolved_dep_groups`. A transaction sender can reference an immature cellbase output as a dep-group cell, bypassing the maturity invariant that every other cell-reference path enforces.

### Finding Description

`ResolvedTransaction` carries three distinct cell collections:

| Field | Role |
|---|---|
| `resolved_inputs` | cells being consumed |
| `resolved_cell_deps` | code/data cells referenced as `Code` deps |
| `resolved_dep_groups` | cells referenced as `DepGroup` deps (expand into further cell deps) |

`MaturityVerifier::verify()` iterates the first two and rejects any immature cellbase cell, but never touches the third:

```rust
// verification/src/transaction_verifier.rs  lines 398-422
if let Some(index) = self.transaction.resolved_inputs.iter().position(cellbase_immature) {
    return Err(TransactionError::CellbaseImmaturity {
        inner: TransactionErrorSource::Inputs, index,
    }.into());
}

if let Some(index) = self.transaction.resolved_cell_deps.iter().position(cellbase_immature) {
    return Err(TransactionError::CellbaseImmaturity {
        inner: TransactionErrorSource::CellDeps, index,
    }.into());
}
// resolved_dep_groups is never checked — Ok(()) returned unconditionally
``` [1](#0-0) 

By contrast, the liveness check in `ResolvedTransaction::check()` does cover all three collections:

```rust
// util/types/src/core/cell.rs  lines 336-376
for cell_meta in &self.resolved_inputs { check_cell(&cell_meta.out_point)?; }
// dep_groups …
for cell_meta in &self.resolved_dep_groups { check_cell(&cell_meta.out_point)?; }
// cell_deps …
for cell_meta in &self.resolved_cell_deps { check_cell(&cell_meta.out_point)?; }
``` [2](#0-1) 

The maturity check is therefore inconsistent with the liveness check: one guard covers all three collections, the other covers only two.

### Impact Explanation

The cellbase maturity rule is a consensus invariant: a miner's coinbase reward output must not be spendable or referenceable until `cellbase_maturity` epochs have elapsed. Allowing an immature cellbase cell to be used as a dep-group violates this invariant. A transaction that passes `MaturityVerifier` but references an immature cellbase dep-group could be admitted to the tx-pool and relayed to peers. If a miner includes such a transaction in a block, nodes that re-verify the block with the same incomplete check would accept it, creating a consensus split between nodes that enforce the full rule and those that do not. Even without block inclusion, the tx-pool admission bypass allows an attacker to pollute the mempool with transactions that reference immature coinbase outputs, undermining the maturity guarantee.

### Likelihood Explanation

No special privilege is required. Any unprivileged RPC caller can submit a transaction via `send_transaction` that references a known immature cellbase output as a `DepGroup` dep. The attacker only needs to know the out-point of a recent (immature) coinbase output, which is publicly visible on-chain. There are no probabilistic or timing prerequisites beyond the normal block production window.

### Recommendation

Add a third check inside `MaturityVerifier::verify()` for `resolved_dep_groups`, mirroring the existing checks for inputs and cell deps:

```rust
if let Some(index) = self
    .transaction
    .resolved_dep_groups
    .iter()
    .position(cellbase_immature)
{
    return Err(TransactionError::CellbaseImmaturity {
        inner: TransactionErrorSource::DepGroups,
        index,
    }
    .into());
}
```

A corresponding `DepGroups` variant should be added to `TransactionErrorSource` if it does not already exist, and a unit test analogous to `test_deps_cellbase_maturity` should be added for dep-groups. [3](#0-2) 

### Proof of Concept

1. Mine block N; its coinbase output `(block_N_hash, 0)` is immature (current epoch < block_N_epoch + cellbase_maturity).
2. Create a cell whose data encodes the out-point `(block_N_hash, 0)` — this makes it a valid dep-group payload.
3. Construct a transaction `T` with a `cell_dep` of type `DepGroup` pointing to that cell.
4. The node resolves `T`: `resolved_dep_groups` contains the immature cellbase cell meta.
5. `MaturityVerifier::verify()` checks `resolved_inputs` (empty) and `resolved_cell_deps` (empty) — both pass. `resolved_dep_groups` is never checked.
6. `T` passes maturity verification and is admitted to the tx-pool, violating the cellbase maturity invariant. [1](#0-0) [4](#0-3)

    pub fn verify(&self) -> Result<(), Error> {
        let cellbase_immature = |meta: &CellMeta| -> bool {
            meta.transaction_info
                .as_ref()
                .map(|info| {
                    info.block_number > 0 && info.is_cellbase() && {
                        let threshold =
                            self.cellbase_maturity.to_rational() + info.block_epoch.to_rational();
                        let current = self.epoch.to_rational();
                        current < threshold
                    }
                })
                .unwrap_or(false)
        };

        if let Some(index) = self
            .transaction
            .resolved_inputs
            .iter()
            .position(cellbase_immature)
        {
            return Err(TransactionError::CellbaseImmaturity {
                inner: TransactionErrorSource::Inputs,
                index,
            }
            .into());
        }

        if let Some(index) = self
            .transaction
            .resolved_cell_deps
            .iter()
            .position(cellbase_immature)
        {
            return Err(TransactionError::CellbaseImmaturity {
                inner: TransactionErrorSource::CellDeps,
                index,
            }
            .into());
        }

        Ok(())
    }

    pub fn check<CC: CellChecker, HC: HeaderChecker, S: BuildHasher>(
        &self,
        seen_inputs: &mut HashSet<OutPoint, S>,
        cell_checker: &CC,
        header_checker: &HC,
    ) -> Result<(), OutPointError> {
        let mut checked_cells: HashSet<OutPoint> = HashSet::new();
        let mut check_cell = |out_point: &OutPoint| -> Result<(), OutPointError> {
            if seen_inputs.contains(out_point) {
                return Err(OutPointError::Dead(out_point.clone()));
            }

            if checked_cells.contains(out_point) {
                return Ok(());
            }

            match cell_checker.is_live(out_point) {
                Some(true) => {
                    checked_cells.insert(out_point.clone());
                    Ok(())
                }
                Some(false) => Err(OutPointError::Dead(out_point.clone())),
                None => Err(OutPointError::Unknown(out_point.clone())),
            }
        };

        // // check input
        for cell_meta in &self.resolved_inputs {
            check_cell(&cell_meta.out_point)?;
        }

        let mut resolved_system_deps: HashSet<&OutPoint> = HashSet::new();
        if let Some(system_cell) = SYSTEM_CELL.get() {
            for cell_meta in &self.resolved_dep_groups {
                let cell_dep = CellDep::new_builder()
                    .out_point(cell_meta.out_point.clone())
                    .dep_type(DepType::DepGroup)
                    .build();

                let dep_group = system_cell.get(&cell_dep);
                if let Some(ResolvedDep::Group(_, cell_deps)) = dep_group {
                    resolved_system_deps.extend(cell_deps.iter().map(|dep| &dep.out_point));
                } else {
                    check_cell(&cell_meta.out_point)?;
                }
            }

            for cell_meta in &self.resolved_cell_deps {
                let cell_dep = CellDep::new_builder()
                    .out_point(cell_meta.out_point.clone())
                    .dep_type(DepType::Code)
                    .build();

                if system_cell.get(&cell_dep).is_none()
                    && !resolved_system_deps.contains(&cell_meta.out_point)
                {
                    check_cell(&cell_meta.out_point)?;
                }
            }
        } else {
            for cell_meta in self
                .resolved_cell_deps
                .iter()
                .chain(self.resolved_dep_groups.iter())
            {
                check_cell(&cell_meta.out_point)?;
            }
        }

        for block_hash in self.transaction.header_deps_iter() {
            header_checker.check_valid(&block_hash)?
        }

        seen_inputs.extend(self.resolved_inputs.iter().map(|i| &i.out_point).cloned());

        Ok(())
    }

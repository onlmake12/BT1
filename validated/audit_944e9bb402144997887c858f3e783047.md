### Title
Unconstrained Provider Fee Parameters in `Echo.sol` Cause `getFee()` to Revert via `SafeCast.toUint96` Overflow, DoS-ing Users of That Provider - (File: target_chains/ethereum/contracts/contracts/echo/Echo.sol)

### Summary
Any permissionlessly registered provider in the Echo contract can set `feePerGasInWei` or `feePerFeedInWei` to a value that causes `getFee()` to revert via a `SafeCast.toUint96` overflow. Because `requestPriceUpdatesWithCallback` calls `getFee()` internally, it also reverts, permanently DoS-ing all users who attempt to request price updates from that provider.

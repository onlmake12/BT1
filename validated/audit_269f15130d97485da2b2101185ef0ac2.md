### Title
Single-Step Ownership Transfer in `PythLazer` Permanently Bricks Trusted Signer Management and Upgrades - (File: `lazer/contracts/evm/src/PythLazer.sol`)

---

### Summary

`PythLazer` inherits from `OwnableUpgradeable` (single-step) instead of `Ownable2StepUpgradeable`. A single admin mistake when calling `transferOwnership` with an incorrect address permanently locks out all `onlyOwner`-gated functions, including trusted signer management and contract upgrades.

---

### Finding Description

`PythLazer` is the on-chain EVM contract for the Pyth Lazer price feed system. It inherits `OwnableUpgradeable` and exposes two `onlyOwner`-gated functions:

- `updateTrustedSigner(address, uint256)` — adds, updates, or removes trusted signers whose ECDSA signatures are verified in `verifyUpdate`
- `_authorizeUpgrade(address)` — gates all UUPS proxy upgrades [1](#0-0) [2](#0-1) 

Because `OwnableUpgradeable` uses a single-step `transferOwnership`, there is no pending-owner confirmation step. If the current owner calls `transferOwnership(wrongAddress)`, ownership is immediately and irrevocably transferred. The new (wrong) address becomes the sole owner, and neither `updateTrustedSigner` nor `_authorizeUpgrade` can ever be called again.

By contrast, every other upgradeable contract in the same repo (`EntropyUpgradable`, `EchoUpgradeable`, `ExecutorUpgradable`, `SchedulerUpgradeable`) already uses `Ownable2StepUpgradeable`: [3](#0-2) [4](#0-3) 

`PythLazer` is the only production contract in scope that still uses the unsafe single-step pattern.

---

### Impact Explanation

**High.** Loss of ownership has two cascading consequences:

1. **Trusted signer management is permanently frozen.** `updateTrustedSigner` is the only way to add, update, or revoke signers. If a signer key is compromised after ownership is lost, it cannot be revoked. All `verifyUpdate` calls signed by the compromised key will continue to pass `isValidSigner` until the signer's `expiresAt` timestamp elapses naturally — potentially for a long time if the signer was set with a far-future expiry. [5](#0-4) 

2. **Contract upgrades are permanently blocked.** `_authorizeUpgrade` is `onlyOwner`, so no bug fixes or security patches can ever be deployed to the proxy. [6](#0-5) 

---

### Likelihood Explanation

**Low.** Requires the owner to supply an incorrect address to `transferOwnership`. This is an admin error, not an attacker-controlled path. However, the impact is permanent and irreversible, which justifies Medium severity (matching the referenced M-02 classification).

---

### Recommendation

Replace `OwnableUpgradeable` with `Ownable2StepUpgradeable` in `PythLazer`, exactly as already done in `EntropyUpgradable`, `EchoUpgradeable`, `ExecutorUpgradable`, and `SchedulerUpgradeable`:

```solidity
// Before
import "@openzeppelin/contracts-upgradeable/access/OwnableUpgradeable.sol";
contract PythLazer is OwnableUpgradeable, UUPSUpgradeable { ... }

// After
import "@openzeppelin/contracts-upgradeable/access/Ownable2StepUpgradeable.sol";
contract PythLazer is Ownable2StepUpgradeable, UUPSUpgradeable { ... }
```

`__Ownable_init` in `initialize` remains compatible — `Ownable2StepUpgradeable` inherits from `OwnableUpgradeable` and reuses the same initializer. [7](#0-6) 

---

### Proof of Concept

1. Owner calls `PythLazer.transferOwnership(0xDEAD...BEEF)` with a typo.
2. `OwnableUpgradeable.transferOwnership` immediately sets `_owner = 0xDEAD...BEEF` with no confirmation step.
3. All subsequent calls to `updateTrustedSigner(...)` revert with `OwnableUnauthorizedAccount`.
4. A previously trusted signer whose key is now compromised cannot be revoked; `isValidSigner(compromisedKey)` returns `true` until its `expiresAt` passes.
5. All calls to `upgradeTo` / `upgradeToAndCall` also revert, permanently blocking any remediation via upgrade. [8](#0-7)

    function _authorizeUpgrade(address) internal override onlyOwner {}

    function updateTrustedSigner(
        address trustedSigner,
        uint256 expiresAt
    ) external onlyOwner {
        if (expiresAt == 0) {
            for (uint8 i = 0; i < trustedSigners.length; i++) {
                if (trustedSigners[i].pubkey == trustedSigner) {
                    trustedSigners[i].pubkey = address(0);
                    trustedSigners[i].expiresAt = 0;
                    delete trustedSignerToExpiresAtMapping[trustedSigner];
                    return;
                }
            }
            revert("no such pubkey");
        } else {
            for (uint8 i = 0; i < trustedSigners.length; i++) {
                if (trustedSigners[i].pubkey == trustedSigner) {
                    trustedSigners[i].expiresAt = expiresAt;
                    trustedSignerToExpiresAtMapping[trustedSigner] = expiresAt;
                    return;
                }
            }
            // Signer not found - adding a new signer.
            for (uint8 i = 0; i < trustedSigners.length; i++) {
                if (trustedSigners[i].pubkey == address(0)) {
                    trustedSigners[i].pubkey = trustedSigner;
                    trustedSigners[i].expiresAt = expiresAt;
                    trustedSignerToExpiresAtMapping[trustedSigner] = expiresAt;
                    return;
                }
            }
            revert("no space for new signer");
        }
    }

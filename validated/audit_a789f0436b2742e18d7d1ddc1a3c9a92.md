### Title
`Scheduler::getPricesNoOlderThan` and `getEmaPricesNoOlderThan` Permanently DoS'd for Whitelisted Subscriptions via Faulty `this.` Internal Call Pattern - (File: `target_chains/ethereum/contracts/contracts/pulse/Scheduler.sol`)

---

### Summary

`Scheduler::getPricesNoOlderThan` and `Scheduler::getEmaPricesNoOlderThan` each verify the caller via the `onlyWhitelistedReader` modifier, then delegate to `this.getPricesUnsafe(...)` / `this.getEmaPricesUnsafe(...)` using an external `this.` call. Because `this.` triggers an external call, `msg.sender` inside the delegated function becomes `address(this)` (the Scheduler contract itself) rather than the original caller. Both `getPricesUnsafe` and `getEmaPricesUnsafe` also carry the `onlyWhitelistedReader` modifier, which will reject `address(this)` as unauthorized whenever a subscription has `whitelistEnabled = true`. The result is that `getPricesNoOlderThan` and `getEmaPricesNoOlderThan` always revert for any whitelist-enabled subscription, permanently breaking these functions for all authorized readers.

---

### Finding Description

`getPricesNoOlderThan` (line 535) and `getEmaPricesNoOlderThan` (line 578) both carry the `onlyWhitelistedReader(subscriptionId)` modifier, which correctly validates the original external caller. [1](#0-0) 

After passing that check, each function delegates to a sibling function using `this.`:

```solidity
prices = this.getPricesUnsafe(subscriptionId, priceIds);   // line 554
prices = this.getEmaPricesUnsafe(subscriptionId, priceIds); // line 597
``` [2](#0-1) [3](#0-2) 

`this.` in Solidity is an external call. The EVM therefore sets `msg.sender = address(this)` inside `getPricesUnsafe` / `getEmaPricesUnsafe`. Both of those functions also carry `onlyWhitelistedReader(subscriptionId)`: [4](#0-3) [5](#0-4) 

The modifier logic is:

```solidity
modifier onlyWhitelistedReader(uint256 subscriptionId) {
    if (_state.subscriptionManager[subscriptionId] == msg.sender) { _; return; }
    if (!_state.subscriptionParams[subscriptionId].whitelistEnabled) { _; return; }
    // linear scan of readerWhitelist for msg.sender …
    if (!isWhitelisted) revert SchedulerErrors.Unauthorized();
    _;
}
``` [6](#0-5) 

When `msg.sender` is `address(this)`:
- The Scheduler contract is never the subscription manager.
- If `whitelistEnabled = false`, the call passes (no bug in that path).
- If `whitelistEnabled = true`, the modifier scans `readerWhitelist` for `address(this)`. The contract address is never in a user-configured whitelist, so `isWhitelisted` remains `false` and the call reverts with `Unauthorized`.

The original caller's authorization (verified in the outer `onlyWhitelistedReader` call) is completely discarded.

---

### Impact Explanation

Any subscription that sets `whitelistEnabled = true` renders `getPricesNoOlderThan` and `getEmaPricesNoOlderThan` permanently non-functional. Every call by any authorized reader — including the subscription manager — reverts with `Unauthorized`. The whitelist feature is specifically designed for protocols that want to restrict price-feed access to known consumers; those are precisely the subscriptions that will be broken. Consumers relying on freshness-bounded reads (`NoOlderThan` variants) cannot obtain price data, breaking downstream protocol logic that depends on these functions.

---

### Likelihood Explanation

Any subscription manager who enables the whitelist (a documented, first-class feature) will trigger this bug. No special attacker action is required; the breakage is automatic and deterministic the moment `whitelistEnabled = true` is set. The likelihood is **high** for any deployment that uses the whitelist feature.

---

### Recommendation

Replace the external `this.` delegation with a direct call to the shared internal helper `_getPricesInternal`, which already exists and carries no access-control modifier:

```diff
 function getPricesNoOlderThan(...) external view override onlyWhitelistedReader(subscriptionId) ... {
     ...
-    prices = this.getPricesUnsafe(subscriptionId, priceIds);
+    PythStructs.PriceFeed[] memory priceFeeds = _getPricesInternal(subscriptionId, priceIds);
+    prices = new PythStructs.Price[](priceFeeds.length);
+    for (uint i = 0; i < priceFeeds.length; i++) { prices[i] = priceFeeds[i].price; }
 }

 function getEmaPricesNoOlderThan(...) external view override onlyWhitelistedReader(subscriptionId) ... {
     ...
-    prices = this.getEmaPricesUnsafe(subscriptionId, priceIds);
+    PythStructs.PriceFeed[] memory priceFeeds = _getPricesInternal(subscriptionId, priceIds);
+    prices = new PythStructs.Price[](priceFeeds.length);
+    for (uint i = 0; i < priceFeeds.length; i++) { prices[i] = priceFeeds[i].emaPrice; }
 }
``` [7](#0-6) 

This eliminates the external re-entry, keeps `msg.sender` as the original caller throughout, and avoids the double whitelist check entirely.

    function _getPricesInternal(
        uint256 subscriptionId,
        bytes32[] calldata priceIds
    ) internal view returns (PythStructs.PriceFeed[] memory priceFeeds) {
        if (!_state.subscriptionParams[subscriptionId].isActive) {
            revert SchedulerErrors.InactiveSubscription();
        }

        SchedulerStructs.SubscriptionParams storage params = _state
            .subscriptionParams[subscriptionId];

        // If no price IDs provided, return all price feeds for the subscription
        if (priceIds.length == 0) {
            PythStructs.PriceFeed[]
                memory allFeeds = new PythStructs.PriceFeed[](
                    params.priceIds.length
                );
            for (uint8 i = 0; i < params.priceIds.length; i++) {
                PythStructs.PriceFeed memory priceFeed = _state.priceUpdates[
                    subscriptionId
                ][params.priceIds[i]];
                // Check if the price feed exists (price ID is valid and has been updated)
                if (priceFeed.id == bytes32(0)) {
                    revert SchedulerErrors.InvalidPriceId(
                        params.priceIds[i],
                        bytes32(0)
                    );
                }
                allFeeds[i] = priceFeed;
            }
            return allFeeds;
        }

        // Return only the requested price feeds
        PythStructs.PriceFeed[]
            memory requestedFeeds = new PythStructs.PriceFeed[](
                priceIds.length
            );
        for (uint8 i = 0; i < priceIds.length; i++) {
            PythStructs.PriceFeed memory priceFeed = _state.priceUpdates[
                subscriptionId
            ][priceIds[i]];

            // Check if the price feed exists (price ID is valid and has been updated)
            if (priceFeed.id == bytes32(0)) {
                revert SchedulerErrors.InvalidPriceId(priceIds[i], bytes32(0));
            }
            requestedFeeds[i] = priceFeed;
        }
        return requestedFeeds;
    }

    modifier onlyWhitelistedReader(uint256 subscriptionId) {
        // Manager is always allowed
        if (_state.subscriptionManager[subscriptionId] == msg.sender) {
            _;
            return;
        }

        // If whitelist is not used, allow any reader
        if (!_state.subscriptionParams[subscriptionId].whitelistEnabled) {
            _;
            return;
        }

        // Check if caller is in whitelist
        address[] storage whitelist = _state
            .subscriptionParams[subscriptionId]
            .readerWhitelist;
        bool isWhitelisted = false;
        for (uint i = 0; i < whitelist.length; i++) {
            if (whitelist[i] == msg.sender) {
                isWhitelisted = true;
                break;
            }
        }

        if (!isWhitelisted) {
            revert SchedulerErrors.Unauthorized();
        }
        _;
    }

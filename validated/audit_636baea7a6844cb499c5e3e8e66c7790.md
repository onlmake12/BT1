### Title
`updateTrustedSigner()` in `PythLazer.sol` Is Controlled by Deployer Owner Rather Than Governance — (`lazer/contracts/evm/src/PythLazer.sol`)

---

### Summary

`PythLazer.sol` gates its critical `updateTrustedSigner()` function with `onlyOwner`. The owner is set at initialization to a `_topAuthority` address — in practice the deployer EOA — rather than the Wormhole governance executor contract. This means the deployer can add, update, or remove trusted Lazer signers entirely outside of governance control, directly analogous to the Infrared report where `toggleSplit()` and `setVoterAndDistributor()` were controlled by the deployer team address instead of the Governor.

---

### Finding Description

`PythLazer.sol` is the EVM Lazer price-verification contract. It maintains a list of `trustedSigners` whose ECDSA signatures are accepted in `verifyUpdate()`. The function that manages this list is:

```solidity
function updateTrustedSigner(
    address trustedSigner,
    uint256 expiresAt
) external onlyOwner { ... }
``` [1](#0-0) 

The owner is set at initialization:

```solidity
function initialize(address _topAuthority) public initializer {
    __Ownable_init(_topAuthority);
``` [2](#0-1) 

The existence of `PythLazerChangeOwnership.s.sol` — a migration script whose sole purpose is to transfer ownership from the deployer EOA to the governance executor — confirms that the deployer holds the owner role at deployment time and must be manually migrated:

> "This script transfers ownership of the deployed PythLazer contract (proxy) to a new owner contract (typically the governance executor contract)." [3](#0-2) 

The contract manager code explicitly acknowledges the design tension:

> `@note The privateKey should be the owner of the Lazer contract. It's not possible to run this function if the executor contract is the owner.` [4](#0-3) 

This means there is a parallel, non-governance code path (`updateTrustedSigner` in `EvmLazerContract`) that calls the function directly with a raw private key, bypassing governance entirely: [5](#0-4) 

The governance path (`generateUpdateTrustedSignerPayload`) only works when the executor IS already the owner: [6](#0-5) 

On any chain where the ownership migration has not yet been executed, the deployer EOA retains unilateral control over trusted signer management.

---

### Impact Explanation

`updateTrustedSigner()` is the sole mechanism for adding or removing trusted Lazer signers. `verifyUpdate()` accepts any update whose ECDSA signature matches a currently-valid trusted signer: [7](#0-6) 

If the deployer owner adds a malicious signer address, all downstream consumers of `verifyUpdate()` will accept fabricated Lazer price payloads signed by that key. This allows arbitrary price manipulation for any protocol that relies on `PythLazer.verifyUpdate()` for price data. Conversely, the owner can remove all legitimate signers, causing a denial-of-service for all Lazer consumers on that chain.

**Severity: Medium** — Trusted signer management is the root of trust for all Lazer price data on EVM chains; compromise allows arbitrary price manipulation.

---

### Likelihood Explanation

**Likelihood: High** — The `PythLazerChangeOwnership.s.sol` script is a post-deployment step that must be run manually per chain. Any chain where this step has not been completed retains the deployer EOA as owner. The TypeScript tooling explicitly provides a direct private-key path for `updateTrustedSigner`, confirming this state is operationally expected on at least some deployments. The deployer key may not maintain the same security posture as the governance executor multisig.

---

### Recommendation

1. Remove the direct `onlyOwner` path for `updateTrustedSigner()` and require all signer management to go through the Wormhole governance executor (i.e., accept a verified VAA, as Sui's `actions.move` already does via `update_trusted_signer`).
2. Ensure the ownership migration (`PythLazerChangeOwnership.s.sol`) is completed on every deployed chain before the contract is used in production.
3. Add a `verification_fee` setter gated by governance, since the fee is currently immutable after initialization. [8](#0-7) 

---

### Proof of Concept

1. Deploy `PythLazer` with `initialize(deployerEOA)`. Owner is `deployerEOA`.
2. Do **not** run `PythLazerChangeOwnership.s.sol` (migration not yet executed).
3. From `deployerEOA`, call:
   ```solidity
   pythLazer.updateTrustedSigner(attackerControlledAddress, block.timestamp + 365 days);
   ```
   This succeeds because `msg.sender == owner()`.
4. Attacker signs any arbitrary Lazer payload with the corresponding private key.
5. Call `pythLazer.verifyUpdate{value: 1 wei}(maliciousUpdate)` — it passes `isValidSigner` and returns the fabricated payload as verified.
6. Any on-chain protocol consuming `verifyUpdate()` accepts the manipulated price data. [9](#0-8) [10](#0-9)

    function updateTrustedSigner(
        address trustedSigner,
        uint256 expiresAt
    ) external onlyOwner {
        if (expiresAt == 0) {
            for (uint8 i = 0; i < trustedSigners.length; i++) {
                if (trustedSigners[i].pubkey == trustedSigner) {
                    trustedSigners[i].pubkey = address(0);
                    trustedSigners[i].expiresAt = 0;
                    delete trustedSignerToExpiresAtMapping[trustedSigner];
                    return;
                }
            }
            revert("no such pubkey");
        } else {
            for (uint8 i = 0; i < trustedSigners.length; i++) {
                if (trustedSigners[i].pubkey == trustedSigner) {
                    trustedSigners[i].expiresAt = expiresAt;
                    trustedSignerToExpiresAtMapping[trustedSigner] = expiresAt;
                    return;
                }
            }
            // Signer not found - adding a new signer.
            for (uint8 i = 0; i < trustedSigners.length; i++) {
                if (trustedSigners[i].pubkey == address(0)) {
                    trustedSigners[i].pubkey = trustedSigner;
                    trustedSigners[i].expiresAt = expiresAt;
                    trustedSignerToExpiresAtMapping[trustedSigner] = expiresAt;
                    return;
                }
            }
            revert("no space for new signer");
        }
    }

    function verifyUpdate(
        bytes calldata update
    ) external payable returns (bytes calldata payload, address signer) {
        // Require fee and refund excess
        require(msg.value >= verification_fee, "Insufficient fee provided");
        if (msg.value > verification_fee) {
            payable(msg.sender).transfer(msg.value - verification_fee);
        }

        if (update.length < 71) {
            revert("input too short");
        }
        uint32 EVM_FORMAT_MAGIC = 706910618;

        uint32 evm_magic = uint32(bytes4(update[0:4]));
        if (evm_magic != EVM_FORMAT_MAGIC) {
            revert("invalid evm magic");
        }
        uint16 payload_len = uint16(bytes2(update[69:71]));
        if (update.length < 71 + payload_len) {
            revert("input too short");
        }
        payload = update[71:71 + payload_len];
        bytes32 hash = keccak256(payload);
        (signer, , ) = ECDSA.tryRecover(
            hash,
            uint8(update[68]) + 27,
            bytes32(update[4:36]),
            bytes32(update[36:68])
        );
        if (signer == address(0)) {
            revert("invalid signature");
        }
        if (!isValidSigner(signer)) {
            revert("invalid signer");
        }
    }

  async updateTrustedSigner(
    trustedSigner: string,
    expiresAt: number,
    privateKey: PrivateKey,
    rawTx = false,
  ): Promise<void> {
    const web3 = this.chain.getWeb3();
    const contract = this.getContract();
    const account = web3.eth.accounts.privateKeyToAccount(privateKey);

    if (rawTx) {
      web3.eth.accounts.wallet.add(privateKey);
      const data = contract.methods
        .updateTrustedSigner(trustedSigner, expiresAt)
        .encodeABI();
      const gasEstimate = await web3.eth.estimateGas({
        data,
        from: account.address,
        to: this.address,
      });
      const tx = await web3.eth.sendTransaction({
        data,
        from: account.address,
        gas: Math.ceil(Number(gasEstimate) * 1.2).toString(),
        to: this.address,
      });
      console.log(
        `Updated trusted signer ${trustedSigner} with expiration ${expiresAt}`,
      );
      console.log(`Transaction hash: ${tx.transactionHash}`);
    } else {
      const gasEstimate = await contract.methods
        .updateTrustedSigner(trustedSigner, expiresAt)
        .estimateGas({ from: account.address });
      const tx = await contract.methods
        .updateTrustedSigner(trustedSigner, expiresAt)
        .send({
          from: account.address,
          gas: Math.floor(gasEstimate * 1.2), // 20% buffer
        });
      console.log(
        `Updated trusted signer ${trustedSigner} with expiration ${expiresAt}`,
      );
      console.log(`Transaction hash: ${tx.transactionHash}`);
    }
  }

public fun update_trusted_signer(state: &mut State, vaa: VAA) {
    let current_cap = state.current_cap();
    let (header, mut parser) = state.unwrap_ptgm(&current_cap, vaa);
    assert!(header.is_update_trusted_signer(), ENotUpdateTrustedSigner);

    let public_key = parser.take_bytes(secp256k1_compressed_pubkey_len());
    let expires_at = parser.take_u64_be();
    parser.destroy_empty();
    state.update_trusted_signer(&current_cap, public_key, expires_at);
}

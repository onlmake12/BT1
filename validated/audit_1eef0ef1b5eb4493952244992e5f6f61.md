### Title
`getPricesNoOlderThan` and `getEmaPricesNoOlderThan` Always Revert for Whitelisted Subscriptions Due to Access Control Mismatch in Self-Call - (`File: target_chains/ethereum/contracts/contracts/pulse/Scheduler.sol`)

---

### Summary

`getPricesNoOlderThan` and `getEmaPricesNoOlderThan` in the Scheduler (Pulse) contract call `this.getPricesUnsafe(...)` and `this.getEmaPricesUnsafe(...)` respectively via an external `this.` call. This causes `msg.sender` inside the callee to be `address(this)` (the Scheduler contract itself), not the original caller. Both callee functions are guarded by `onlyWhitelistedReader`, which rejects `address(this)` because the contract is never the subscription manager and is never in any reader whitelist. As a result, both staleness-checked read functions unconditionally revert for any subscription with `whitelistEnabled = true`.

---

### Finding Description

`getPricesNoOlderThan` (line 535) and `getEmaPricesNoOlderThan` (line 578) are `external view` functions protected by `onlyWhitelistedReader`. After performing their staleness check, they delegate to `this.getPricesUnsafe(...)` and `this.getEmaPricesUnsafe(...)` respectively:

```solidity
// Scheduler.sol line 554
prices = this.getPricesUnsafe(subscriptionId, priceIds);

// Scheduler.sol line 597
prices = this.getEmaPricesUnsafe(subscriptionId, priceIds);
```

Because `this.` triggers an external call, `msg.sender` inside `getPricesUnsafe` / `getEmaPricesUnsafe` becomes `address(this)` — the Scheduler contract itself — not the original external caller.

Both `getPricesUnsafe` and `getEmaPricesUnsafe` are also guarded by `onlyWhitelistedReader`:

```solidity
modifier onlyWhitelistedReader(uint256 subscriptionId) {
    if (_state.subscriptionManager[subscriptionId] == msg.sender) { _; return; }
    if (!_state.subscriptionParams[subscriptionId].whitelistEnabled) { _; return; }
    // whitelist scan ...
    if (!isWhitelisted) { revert SchedulerErrors.Unauthorized(); }
    _;
}
```

When `msg.sender` is `address(this)`:
- The Scheduler contract is never the `subscriptionManager` (that is always the user who called `createSubscription`).
- If `whitelistEnabled = true`, the Scheduler contract is never in the `readerWhitelist`.
- The modifier therefore always reverts with `Unauthorized`.

The result: **`getPricesNoOlderThan` and `getEmaPricesNoOlderThan` are completely non-functional for any subscription with `whitelistEnabled = true`**, regardless of whether the original caller is the manager or a legitimately whitelisted reader. [1](#0-0) [2](#0-1) [3](#0-2) 

---

### Impact Explanation

The Pulse/Scheduler README explicitly recommends `getPricesNoOlderThan` and `getEmaPricesNoOlderThan` as the safe, staleness-checked variants for consuming prices. Whitelisted subscriptions are a first-class feature — the `whitelistEnabled` flag exists precisely for protocols that want to restrict who can read their subscription's prices.

For any such subscription:
- The recommended safe read path (`getPricesNoOlderThan` / `getEmaPricesNoOlderThan`) is permanently broken.
- Readers are forced to fall back to `getPricesUnsafe` / `getEmaPricesUnsafe`, which provide no staleness guarantee, defeating the purpose of the safer API.
- Any on-chain consumer contract that calls `getPricesNoOlderThan` on a whitelisted subscription will always revert, breaking integrations. [4](#0-3) 

---

### Likelihood Explanation

`whitelistEnabled` is a documented, user-facing feature. Any subscription manager who enables it (e.g., a protocol restricting price reads to its own contracts) will trigger this bug the moment any reader calls `getPricesNoOlderThan`. The bug is deterministic and 100% reproducible — it requires no special conditions beyond `whitelistEnabled = true`. The attacker-controlled entry path is simply calling `getPricesNoOlderThan` on any whitelisted subscription. [5](#0-4) [6](#0-5) 

---

### Recommendation

Replace the external `this.` calls with direct calls to the internal helper `_getPricesInternal`, which has no access control:

```solidity
// Before (broken):
prices = this.getPricesUnsafe(subscriptionId, priceIds);

// After (fixed):
PythStructs.PriceFeed[] memory priceFeeds = _getPricesInternal(subscriptionId, priceIds);
prices = new PythStructs.Price[](priceFeeds.length);
for (uint i = 0; i < priceFeeds.length; i++) {
    prices[i] = priceFeeds[i].price; // or .emaPrice for EMA variant
}
```

Apply the same fix to `getEmaPricesNoOlderThan`. The `onlyWhitelistedReader` guard on the outer function already ensures only authorized callers reach this code path. [7](#0-6) 

---

### Proof of Concept

1. Deploy `SchedulerUpgradeable` and create a subscription with `whitelistEnabled = true` and a non-empty `readerWhitelist` containing address `0xAlice`.
2. Call `updatePriceFeeds` to populate price data.
3. Call `getPricesNoOlderThan(subscriptionId, priceIds, 300)` from `0xAlice` (a legitimately whitelisted reader).
4. Execution flow:
   - `onlyWhitelistedReader` on `getPricesNoOlderThan`: `msg.sender = 0xAlice` → passes (Alice is whitelisted).
   - Staleness check passes.
   - `this.getPricesUnsafe(subscriptionId, priceIds)` is called → external call → `msg.sender` inside `getPricesUnsafe` = `address(Scheduler)`.
   - `onlyWhitelistedReader` on `getPricesUnsafe`: `address(Scheduler)` is not the manager, whitelist is enabled, `address(Scheduler)` is not in whitelist → **reverts with `Unauthorized`**.
5. The transaction reverts despite Alice being a legitimately whitelisted reader. [8](#0-7) [9](#0-8)

    function _getPricesInternal(
        uint256 subscriptionId,
        bytes32[] calldata priceIds
    ) internal view returns (PythStructs.PriceFeed[] memory priceFeeds) {
        if (!_state.subscriptionParams[subscriptionId].isActive) {
            revert SchedulerErrors.InactiveSubscription();
        }

        SchedulerStructs.SubscriptionParams storage params = _state
            .subscriptionParams[subscriptionId];

        // If no price IDs provided, return all price feeds for the subscription
        if (priceIds.length == 0) {
            PythStructs.PriceFeed[]
                memory allFeeds = new PythStructs.PriceFeed[](
                    params.priceIds.length
                );
            for (uint8 i = 0; i < params.priceIds.length; i++) {
                PythStructs.PriceFeed memory priceFeed = _state.priceUpdates[
                    subscriptionId
                ][params.priceIds[i]];
                // Check if the price feed exists (price ID is valid and has been updated)
                if (priceFeed.id == bytes32(0)) {
                    revert SchedulerErrors.InvalidPriceId(
                        params.priceIds[i],
                        bytes32(0)
                    );
                }
                allFeeds[i] = priceFeed;
            }
            return allFeeds;
        }

        // Return only the requested price feeds
        PythStructs.PriceFeed[]
            memory requestedFeeds = new PythStructs.PriceFeed[](
                priceIds.length
            );
        for (uint8 i = 0; i < priceIds.length; i++) {
            PythStructs.PriceFeed memory priceFeed = _state.priceUpdates[
                subscriptionId
            ][priceIds[i]];

            // Check if the price feed exists (price ID is valid and has been updated)
            if (priceFeed.id == bytes32(0)) {
                revert SchedulerErrors.InvalidPriceId(priceIds[i], bytes32(0));
            }
            requestedFeeds[i] = priceFeed;
        }
        return requestedFeeds;
    }

        uint256 subscriptionId,
        bytes32[] calldata priceIds,
        uint256 age_seconds
    )
        external
        view
        override
        onlyWhitelistedReader(subscriptionId)
        returns (PythStructs.Price[] memory prices)
    {
        SchedulerStructs.SubscriptionStatus memory status = _state
            .subscriptionStatuses[subscriptionId];

        // Use distance (absolute difference) since pythnet timestamps
        // may be slightly ahead of this chain.
        if (distance(block.timestamp, status.priceLastUpdatedAt) > age_seconds)
            revert PythErrors.StalePrice();

        prices = this.getPricesUnsafe(subscriptionId, priceIds);
    }

    modifier onlyWhitelistedReader(uint256 subscriptionId) {
        // Manager is always allowed
        if (_state.subscriptionManager[subscriptionId] == msg.sender) {
            _;
            return;
        }

        // If whitelist is not used, allow any reader
        if (!_state.subscriptionParams[subscriptionId].whitelistEnabled) {
            _;
            return;
        }

        // Check if caller is in whitelist
        address[] storage whitelist = _state
            .subscriptionParams[subscriptionId]
            .readerWhitelist;
        bool isWhitelisted = false;
        for (uint i = 0; i < whitelist.length; i++) {
            if (whitelist[i] == msg.sender) {
                isWhitelisted = true;
                break;
            }
        }

        if (!isWhitelisted) {
            revert SchedulerErrors.Unauthorized();
        }
        _;
    }

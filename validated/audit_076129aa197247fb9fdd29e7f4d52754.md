### Title
Unbounded Cell Scan in `get_cells_capacity` Causes RPC DoS — (`util/indexer/src/service.rs`)

### Summary

`IndexerHandle::get_cells_capacity()` iterates over every matching live cell in the RocksDB store without enforcing the `request_limit` count cap that `get_cells()` and `get_transactions()` both enforce. The only protection is a wall-clock `TimeoutIterator` (default 10 s). Any unprivileged RPC caller can issue repeated broad-prefix `get_cells_capacity` requests that each saturate the indexer thread for the full timeout window, causing sustained CPU/IO exhaustion and degrading or blocking all other indexer RPC service.

### Finding Description

`get_cells()` and `get_transactions()` both gate their RocksDB scans with an explicit count limit:

```rust
// util/indexer/src/service.rs  ~line 216
if limit > self.request_limit {
    return Err(Error::invalid_params(format!(
        "limit must be less than {}",
        self.request_limit,
    )));
}
``` [1](#0-0) 

`get_cells_capacity()` has no equivalent guard. It opens a RocksDB iterator and consumes every key that matches the caller-supplied prefix:

```rust
// util/indexer/src/service.rs  ~line 726
let capacity: u64 = iter
    .by_ref()
    .take_while(|(key, _value)| key.starts_with(&prefix))
    .filter_map(|(key, value)| { … })
    .sum();
``` [2](#0-1) 

The only bound is `TimeoutIterator`, which stops iteration after `timeout_limit` seconds (default 10 s, from `config.timeout_limit.unwrap_or(10)`): [3](#0-2) 

The `request_limit` field itself defaults to `usize::MAX` when unconfigured: [4](#0-3) 

The RPC surface is publicly documented and requires no authentication: [5](#0-4) 

**Exploit path:**

1. Attacker identifies (or creates) a `code_hash` that is used by many live cells on-chain (e.g., the standard secp256k1 lock).
2. Attacker calls `get_cells_capacity` with `script_type: lock`, `script_search_mode: prefix`, and `args: "0x"` (empty args → matches every cell locked by that `code_hash`).
3. The indexer scans millions of RocksDB entries for the full 10-second timeout window per request.
4. Attacker sends N concurrent requests; each occupies the indexer for 10 s of CPU/IO.
5. Legitimate `get_cells`, `get_transactions`, and `get_cells_capacity` calls are starved or time out.

No privileged access, no key, no special role is required — only the ability to reach the RPC port.

### Impact Explanation

Sustained CPU and RocksDB I/O exhaustion on the CKB node's indexer process. All indexer RPC methods (`get_cells`, `get_transactions`, `get_cells_capacity`) share the same single-threaded or thread-pool-limited execution context. Flooding `get_cells_capacity` with broad-prefix queries degrades or completely blocks indexer RPC availability for the duration of the attack. Nodes that expose the indexer RPC publicly (common for infrastructure providers and wallets) are directly affected.

### Likelihood Explanation

The RPC endpoint is publicly documented and requires no credentials. The attack requires only knowledge of a popular `code_hash` (trivially obtained from the chain) and the ability to send HTTP JSON-RPC requests. The default `timeout_limit` of 10 s and the absence of any per-IP or per-method rate limiting make sustained flooding straightforward.

### Recommendation

Apply the same `request_limit` guard used by `get_cells` and `get_transactions` to `get_cells_capacity`. Because `get_cells_capacity` is an aggregate (no pagination), the appropriate mitigation is to add a configurable maximum-scan-count that aborts and returns an error when exceeded, analogous to how `get_cells` rejects requests whose `limit` exceeds `self.request_limit`. Additionally, consider adding a per-IP or per-method rate limit at the RPC layer.

### Proof of Concept

```json
// Repeated N times concurrently:
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "get_cells_capacity",
  "params": [{
    "script": {
      "code_hash": "0x9bd7e06f3ecf4be0f2fcd2188b23f1b9fcc88e5d4b65a8637b17723bbda3cce8",
      "hash_type": "type",
      "args": "0x"
    },
    "script_type": "lock",
    "script_search_mode": "prefix"
  }]
}
```

`args: "0x"` (empty) with `prefix` mode matches every cell whose lock script uses the given `code_hash`, forcing the indexer to scan the entire matching key space in RocksDB for up to `timeout_limit` seconds per request. With N concurrent requests the indexer is continuously saturated.

    pub fn get_cells_capacity(
        &self,
        search_key: IndexerSearchKey,
    ) -> Result<Option<IndexerCellsCapacity>, Error> {
        if search_key
            .script_search_mode
            .as_ref()
            .map(|mode| *mode == IndexerSearchMode::Partial)
            .unwrap_or(false)
        {
            return Err(Error::invalid_params(
                "the CKB indexer doesn't support search_key.script_search_mode partial search mode, \
                please use the CKB rich-indexer for such search",
            ));
        }

        let (prefix, from_key, direction, skip) = build_query_options(
            &search_key,
            KeyPrefix::CellLockScript,
            KeyPrefix::CellTypeScript,
            IndexerOrder::Asc,
            None,
        )?;
        let filter_script_type = match search_key.script_type {
            IndexerScriptType::Lock => IndexerScriptType::Type,
            IndexerScriptType::Type => IndexerScriptType::Lock,
        };
        let script_search_exact = matches!(
            search_key.script_search_mode,
            Some(IndexerSearchMode::Exact)
        );
        let filter_options: FilterOptions = search_key.try_into()?;
        let mode = IteratorMode::From(from_key.as_ref(), direction);
        let snapshot = self.store.inner().snapshot();
        let mut iter = TimeoutIterator::new(snapshot.iterator(mode).skip(skip), self.timeout_limit);
        let pool = self
            .pool
            .as_ref()
            .map(|pool| pool.read().expect("acquire lock"));

        let capacity: u64 = iter
            .by_ref()
            .take_while(|(key, _value)| key.starts_with(&prefix))
            .filter_map(|(key, value)| {
                if script_search_exact {
                    // Exact match mode, check key length is equal to full script len + BlockNumber (8) + TxIndex (4) + OutputIndex (4)
                    if key.len() != prefix.len() + 16 {
                        return None;
                    }
                }
                let tx_hash = packed::Byte32::from_slice(value.as_ref()).expect("stored tx hash");
                let index =
                    u32::from_be_bytes(key[key.len() - 4..].try_into().expect("stored index"));
                let out_point = packed::OutPoint::new(tx_hash, index);
                if pool
                    .as_ref()
                    .map(|pool| pool.is_consumed_by_pool_tx(&out_point))
                    .unwrap_or_default()
                {
                    return None;
                }
                let (block_number, _tx_index, output, output_data) = Value::parse_cell_value(
                    &snapshot
                        .get(Key::OutPoint(&out_point).into_vec())
                        .expect("get OutPoint should be OK")
                        .expect("stored OutPoint"),
                );

                if let Some(prefix) = filter_options.script_prefix.as_ref() {
                    match filter_script_type {
                        IndexerScriptType::Lock => {
                            if !extract_raw_data(&output.lock())
                                .as_slice()
                                .starts_with(prefix)
                            {
                                return None;
                            }
                        }
                        IndexerScriptType::Type => {
                            if output.type_().is_none()
                                || !extract_raw_data(&output.type_().to_opt().unwrap())
                                    .as_slice()
                                    .starts_with(prefix)
                            {
                                return None;
                            }
                        }
                    }
                }

                if let Some([r0, r1]) = filter_options.script_len_range {
                    match filter_script_type {
                        IndexerScriptType::Lock => {
                            let script_len = extract_raw_data(&output.lock()).len();
                            if script_len < r0 || script_len > r1 {
                                return None;
                            }
                        }
                        IndexerScriptType::Type => {
                            let script_len = output
                                .type_()
                                .to_opt()
                                .map(|script| extract_raw_data(&script).len())
                                .unwrap_or_default();
                            if script_len < r0 || script_len > r1 {
                                return None;
                            }
                        }
                    }
                }

                if let Some((data, mode)) = &filter_options.output_data {
                    match mode {
                        IndexerSearchMode::Prefix => {
                            if !output_data.raw_data().starts_with(data) {
                                return None;
                            }
                        }
                        IndexerSearchMode::Exact => {
                            if output_data.raw_data() != data {
                                return None;
                            }
                        }
                        IndexerSearchMode::Partial => {
                            memmem::find(&output_data.raw_data(), data)?;
                        }
                    }
                }

                if let Some([r0, r1]) = filter_options.output_data_len_range
                    && (output_data.len() < r0 || output_data.len() >= r1)
                {
                    return None;
                }

                if let Some([r0, r1]) = filter_options.output_capacity_range {
                    let capacity: core::Capacity = output.capacity().into();
                    if capacity < r0 || capacity >= r1 {
                        return None;
                    }
                }

                if let Some([r0, r1]) = filter_options.block_range
                    && (block_number < r0 || block_number >= r1)
                {
                    return None;
                }

                Some(Into::<core::Capacity>::into(output.capacity()).as_u64())
            })
            .sum();

### Title
Missing Replay Protection in `verifyUpdate` Allows Indefinite Reuse of Captured Signed Updates — (`lazer/contracts/evm/src/PythLazer.sol`)

### Summary
`PythLazer.verifyUpdate` is entirely stateless. It performs no tracking of previously accepted signatures, so any captured `(r, s, v, payload)` triple can be submitted verbatim in any future block and will succeed unconditionally as long as the signer has not expired.

### Finding Description
`verifyUpdate` performs exactly three checks before returning `(payload, signer)` to the caller:

1. `msg.value >= verification_fee` — fee guard
2. EVM format magic byte match
3. `isValidSigner(signer)` — checks only `block.timestamp < trustedSignerToExpiresAtMapping[signer]` [1](#0-0) 

No state is written on success. There is no `mapping(bytes32 => bool) usedHashes`, no nonce, and no enforcement of the `timestamp` field embedded in the payload. The function is a pure read-and-verify primitive with no memory of prior calls. [2](#0-1) 

The payload does carry a `timestamp` field (parsed by `PythLazerLib.parsePayloadHeader`), but `verifyUpdate` never reads or enforces it. [3](#0-2) 

### Impact Explanation
Any consumer contract that calls `verifyUpdate` as its sole security boundary — which is the intended usage pattern — is vulnerable to replay. An attacker who observes a valid update in block N can re-submit the identical calldata in block N+M. `verifyUpdate` returns the same `(payload, signer)` pair, and the consumer processes a stale price as if it were fresh. The captured signature remains permanently valid until the signer's `expiresAt` timestamp passes, which can be months or years in the future. [4](#0-3) 

### Likelihood Explanation
The attack requires only: (1) monitoring on-chain calldata for a prior `verifyUpdate` call, and (2) re-submitting the same bytes with `msg.value >= 1 wei`. No privileged access, no key material, and no off-chain infrastructure are needed. Any public mempool observer can execute this.

### Recommendation
Add a `mapping(bytes32 => bool) private usedPayloadHashes` to `PythLazer`. In `verifyUpdate`, after computing `hash = keccak256(payload)`, assert `!usedPayloadHashes[hash]`, then set `usedPayloadHashes[hash] = true`. Additionally, enforce a maximum age on the payload's embedded `timestamp` (e.g., reject if `payload_timestamp < block.timestamp - MAX_AGE`) to bound the replay window even before a hash is recorded.

### Proof of Concept
```solidity
// 1. Deploy PythLazer, register a trusted signer, craft a valid update.
bytes memory update = buildValidUpdate(signerKey, payload);

// 2. First call — succeeds.
(bytes memory p1, address s1) = pythLazer.verifyUpdate{value: 1 wei}(update);

// 3. Second call with identical bytes in a later block — also succeeds.
(bytes memory p2, address s2) = pythLazer.verifyUpdate{value: 1 wei}(update);

// p1 == p2, s1 == s2 — no revert, no state change, full replay accepted.
assert(keccak256(p1) == keccak256(p2));
``` [1](#0-0)

    function verifyUpdate(
        bytes calldata update
    ) external payable returns (bytes calldata payload, address signer) {
        // Require fee and refund excess
        require(msg.value >= verification_fee, "Insufficient fee provided");
        if (msg.value > verification_fee) {
            payable(msg.sender).transfer(msg.value - verification_fee);
        }

        if (update.length < 71) {
            revert("input too short");
        }
        uint32 EVM_FORMAT_MAGIC = 706910618;

        uint32 evm_magic = uint32(bytes4(update[0:4]));
        if (evm_magic != EVM_FORMAT_MAGIC) {
            revert("invalid evm magic");
        }
        uint16 payload_len = uint16(bytes2(update[69:71]));
        if (update.length < 71 + payload_len) {
            revert("input too short");
        }
        payload = update[71:71 + payload_len];
        bytes32 hash = keccak256(payload);
        (signer, , ) = ECDSA.tryRecover(
            hash,
            uint8(update[68]) + 27,
            bytes32(update[4:36]),
            bytes32(update[36:68])
        );
        if (signer == address(0)) {
            revert("invalid signature");
        }
        if (!isValidSigner(signer)) {
            revert("invalid signer");
        }
    }
